README

Included:
Python sample commands which contains some common scripts that allow you to run Linux commands via SSH. 
You can customize the scripts, as required, in order to run procedures on CloudShell resources

# Execute Procedure.py
Executes predefined procedures saved to a file which is accessible to the Execution Server.

# Execute Command.py
Runs any command.

# Grep.py
Searches for a specific string in a file.

# ls.py
Lists all the files and folders in the current directory of the resource.
